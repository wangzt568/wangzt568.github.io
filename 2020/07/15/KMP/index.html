Algo-KMP

KMP算法，全称Knuth-Morris-Pratt算法，根据三个作者Donald Knuth、Vaughan Pratt、James H. Morris的姓氏的首字母拼接而成的。是一种字符串匹配的算法，用于在一个文本串S中查找模式串P的位置。

核心思想, 建立数组存储模式串pattern 当前比较位置之前的substring里的相同的prefix和surfix的最长长度. 如果P和S不匹配,

S指针无需跳到起始位置. P指针跳到prefix之后一位,继续进行比较.

详见代码 

public List<String> stringMatching(String[] words) {
          List<String> list = new ArrayList<String>();
          for(int i = 0; i < words.length; i++) {
              for(int j = 0; j < words.length; j++){
                  if(j != i && words[i].length() <= words[j].length() && KMPmatch(words[i], words[j]) && !list.contains(words[i])){
                      list.add(words[i]);
                  }
              }
          }
          return list;
      }
      
      public int[] samepresurfix(String s){
          int [] sublen = new int[s.length()];
          int index = 0;
          int i = 1;
            while(i < s.length()){
              if(s.charAt(index) == s.charAt(i)){
                  sublen[i] = index + 1;
                  i++;
                  index++;
              } else {
                  if(index != 0) {
                      index = sublen[index - 1];
                  } else {
                      sublen[i] = 0;
                      i++;
                  }                             
              }
          }
          for(int k = 0; k < sublen.length; k++) {
              System.out.print(sublen[k] + " ");
          }
          return sublen;
      }
      
      public boolean KMPmatch (String p, String s) {
          int [] sublen = samepresurfix(p);
          int i = 0;
          int j = 0;
          while(j < p.length() && i < s.length()) {
              if(p.charAt(j) == s.charAt(i)){
                  i++;
                  j++;
              } else {
                  if(j != 0) {
                      j = sublen[j - 1];
                  } else {
                      i++;
                  }      
              }
          }
          if(j == p.length()) {
              return true;
          }
          
          return false;
      }

Runtime从brute force方法的O(m * n) 减少到O (m + n);

Extra Space: O(n)

//n为Pattern 的长度
